# U.S. Name Change Requirements by State (open dataset)

A free, open, source-verified dataset of the driver's license / state ID name-change requirements after marriage across 46 U.S. jurisdictions. Each row lists the issuing agency, the fee, whether an in-person visit is required, the legal deadline (or a note if none is published), the official government source URL, and the date the row was last verified.

## Files
- `name-change-requirements-by-state.csv`

## What the data shows
Of the 46 jurisdictions checked, 40 require an in-person office visit, only 18 publish a deadline, and fees range from $3 to $42.50. Every row links to its official state source.
